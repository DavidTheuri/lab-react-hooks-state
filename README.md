# React Shopping App

This is a simple shopping app built with React and the useState hook.  
The project was created as part of a React Hooks lab assignment to practice managing state, handling events, and working with components.

## Features

- Toggle between dark mode and light mode
- Filter products by category
- Add products to the shopping cart
- View cart items dynamically

## Technologies Used

- React
- JavaScript
- Vite
- CSS

## Getting Started

Clone the repository:

```bash
git clone https://github.com/DavidTheuri/lab-react-hooks-state.git
```

Move into the project folder:

```bash
cd lab-react-hooks-state
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
